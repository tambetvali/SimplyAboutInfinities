# This folder lists the basic simulation files, but you also need to know where more of these files are located:

Open source [Sheep Counters](https://spireason.neocities.org/#sheep) contain json, data and generational files for number database, and files here are selected by containing everything in a few files, from there.

[LaeLane](https://github.com/tambetvali/LaeLane/) is next project for number simulations:
- [SrcAIPnP](https://github.com/tambetvali/LaeLane/tree/main/LaegnaLane/SrcAIPnP) - another folder *very similar with the one here*, also containing the same files, but additionally some driver files or generators are added so that more advanced AI could get more out of it.
  - Yet, this folder here is *minimalist* and either three json files, or the single multiline.md would fit better to AI context, especially the latter.
- [CanvasMode](https://github.com/tambetvali/LaeLane/tree/main/CanvasMode) - this will be the *only* simulation mode data folder which does not repeat the files from sheep counters by choosing right subsets; CanvasMode files will contain data about how I draw Laegna Numbers using coordinates, so that you strictly get the single right standard angle for example, or a straight line.
