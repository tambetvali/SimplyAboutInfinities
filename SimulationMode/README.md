At this website section, either user can animate this (json is rather for robots and AIs), or one can download generators: with generators, you can find single numbers to change, to generate numbers with more digits in all three systems, and achieve larger json files, possible extensions to Laegna Number Database. An AI could use the code files to confirm structure it learns from examples, and it's applicability to non-examplified numbers which come beyond.

Number Data files have linear, discrete, naturally ordered version of Laegna numbers, and static boundary positions:
- This is designed specifically for ordinal-level logic and math, and initial study, when more complex Laegna systems were already applied to it's domains of thinking and math.
- Typically, same boundary letters are used differently in each number system, but the version contained is somewhat reasonable for both signed and unsigned numbers.
- Typically, Laegna number is two-dimensional, but ordering given allows the simplest linear transformation to it's meanings accross theories, thus it's kind of "true" ordinal meaning.
- Educationally: Sheep Counter is intented to be parallel to learning numbers in each language, where you typically count Natural Numbers from zero or one, to some degree such as 1000, and if this is your high-alarm mode mentally prepared, you learn typical system in one such counting.
  - Educationally, Sheep Counter tries to give similar experience.
  - Json files contain information for animation, timing and numbers with their meanings. Animations, games, and browsers can be produced easily, even with one CoPilot session as single, not very large HTML file for basic criteria of truth and applicability: html, also, can use json file dynamically to avoid copying errors and dramatic sizes of intelligent data to handle (time and space) complexity.
- Computationally:
  - Because implementing whole theorem framework is very complex,
  - simple set of numbers can be used to emulate operations,
  - json files can thus be used directly or indirectly to produce basic data for introductionary math, used in game, emulation or creation of small sets of operation mappings, such as primitive table of multiplication - digitwise, from I to E, 16 cells are contained in complete table of multiplication, and thus you use R = {1, 2}, namely one- and two-digit length numbers to confirm each form and understand the symmetries and hidden implications such as inside, there might be *one or two digit number* or *only two digit numbers*, different symmetrically in some applications and their particular definitions.
